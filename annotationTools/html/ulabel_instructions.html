The Guide to uLabel

The process of annotation is an ardous process that is frustrating if not done with the right tool. Hopefully, uLabel will ease your pain here. Get better and faster at it by learning and getting use to the various shortcuts implemented. So, watch out for all the shortcuts as you learn in this guide.

In this guide, you will learn:

  1. How to navigate
  2. How to label
  3. How to check progress
  4. How to edit
  5. Summary of shortucts

1. How to navigate

Two ways: 1) Changing URL 2) By nagivational arrows

First, get the training set folder name and the first image name from the DSO team. Then, you will access the tool with the following URL with a query string at the end of it: http://website.com/ulabel/tool.html?folder=folder_name&image=image_name00001.jpg change the parameters of the query string accordingly for the "folder" and "image".

To access the next training image you can explicitly redefine the next image name in the query string of the URL.

Alternatively, (reccomended) use the navigational arrows at the top bar to get to the next or previous training image. This will run through the images by alphabetical order.

To zoom in on the image, simply press '1' for zoom in, '2' for zoom out, and '3' for completely zoom out.
 

2. How to label

Two ways - 1) Polygon, 2) Scribble

For beginners, (reccomended) just use polygon. Here, I will describe the polygon process. Scribble function is as instinctive as it can get, just try it out and you will get it. Whether or not you find it useful is up to one's preference.

Polygon plotting:

  1. Identify boundary/contour of the feature to be annotated.
  2. Dot vertices of the boundary using mouse left click.
  3. Make sure polygon plotted has a good fit to the actual contour of the feature.
  4. On the last point, you can either right click or press back on the first point to complete.
  5. Select appropriate class (here you can just type the first letter of the class to select it).
  6. Done! (alternatively press 'd')

Other features:

  • Opps (or just press 'z') can be select when you accidentally end the polygon and want to go back to select it.
  • Delete (or just press'`', this key is found just below 'esc') when you just want to delete the plotted polygon
  • When plotting polygon, simply press 'z' if you make a mistake plotting the current dot it will undo the last move. (or, the undo button can be manually pressed too)
  • You can release the held outlines by selecting 'Hide all' from the side, if you find it obstructing your plotting

What is an appropriate class:

  1. Sky: simply the sky
  2. Road: traffic roads
  3. White Road Marking: white arrows, lane markings, etc. found on the road
  4. Yellow Road Marking: yellow box, double yellow line, etc.
  5. Kerb: side of the road
  6. Pavement: areas where pedestrian belongs
  7. Low Vegetation: any grass and bushes.
  8. High Vegetation: all other plants that are not low vegetation.
  9. Tree trunk: big and obvious tree trunks.
  10. Building: man-made concrete structures, I would consider bridge buildings as well.
  11. Object: all other obstacles/signs/objects, like cars, humans, traffic lights, road signs, obstructions.
  12. Pole: poles holding up signs, traffic lights, street lamps, etc.
 

3. How to check progress

  1. Once you have annotated a few features,
  2. Press 's' (show)
  3. Be amazed
  4. Press 'h' (hide)
 

4. How to edit

Click on the polygon or select from object list.

Useful feature: you can adjust the polygon vertices even after input!
